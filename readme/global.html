Neighborhood Map Project - Providence, RI Created using the Google Maps API Location data from Foursquare Weather data from Weather Underground To run the program, clone the Github repository to your machine, open the dist folder and run index.html in the browser of your choice This app allows you to: Select a location to view contact information and link to its website Browse for other nearby locations using the "Browse Nearby!" button Filter hotspots by typing in the search bar
